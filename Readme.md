# Handwritten Digit Recognizer using CNN from Scratch




### An end-to-end handwritten digit recognition application built entirely from scratch using NumPy. The project implements a Convolutional Neural Network (CNN) without deep learning frameworks and provides an interactive GUI for drawing and predicting digits.
#
# Features

### ✅ CNN implemented from scratch using NumPy

### ✅ Convolution and Fully Connected layers

### ✅ ReLU and Softmax activations

### ✅ Cross-Entropy Loss and Backpropagation

### ✅ Interactive Tkinter drawing canvas

### ✅ Real-time digit prediction and confidence score

### ✅ User feedback collection for incorrect predictions

### ✅ Personalized retraining using custom handwriting samples

### ✅ Model weight saving and loading
